## 1 next/linkとuseRouterの違い
`next/link`はUI上にリンクを作る。html`<a>`のNext.js版
```
import Link from "next/link";

<Link href="/about">About</Link>
```
`useRouter` は処理が終わった後、コードから移動させたいときにつかる。
 - フォーム送信後
 - ログイン後 など
```
import { useRouter } from "next/navigation";

const router = useRourter();
router.push("/about");
```
